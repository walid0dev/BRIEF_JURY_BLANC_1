- $match : filters documents by condition
- $lookup : joins documents from another collection
- $expr : allows aggregation expressions inside a query filter
- $eq : checks if two values are equal
- $group : groups documents and computes aggregate values
- $sum : adds numeric values across documents or arrays
- $addFields : adds or overwrites fields in each document
- $size : returns the number of elements in an array
- $ifNull : returns a fallback value when input is null or missing
- $first : returns the first value from an array or grouped set
- $cond : applies if/then/else conditional logic
- $gt : checks if one value is greater than another
- $round : rounds a number to a specified precision
- $multiply : multiplies numeric values
- $divide : divides one numeric value by another
- $arrayToObject : converts key-value pairs into an object
- $map : transforms each element of an array
- $filter : keeps only array elements that match a condition
- $subtract : subtracts one value from another
- $project : includes, excludes, or reshapes fields in output documents
